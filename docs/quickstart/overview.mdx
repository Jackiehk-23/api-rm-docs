Set up authentication and make your first RM API call. All endpoints require a valid HMAC-SHA256 signature and an access token — start here.

<Admonition type="tip" title="Recommended Order">
1. Read [Signature Algorithm](./signature-algorithm) — required for every request.
2. Get an [Access Token](./accesstoken/client-credentials).
3. Make your first API call.
</Admonition>
