## Let's start call the API

**Method:** <HttpMethodBadge method="POST" />

URL: `https://open.revenuemonster.my/v3/service`

Sandbox URL: `https://sb-open.revenuemonster.my/v3/service`

### Header Request

| Key             | Value                            | Description                                                           |
| --------------- | -------------------------------- | --------------------------------------------------------------------- |
| `Content-Type`  | application/json                 | Content Type                                                          |
| `Authorization` | Bearer \{Token\}                   | Refer to [Access Token](../quickstart/accesstoken/client-credentials) |
| `X-Signature`   | sha256 \{Signature\}               | Refer to [Signature Algorithm](../quickstart/signature-algorithm)     |
| `X-Nonce-Str`   | VYNknZohxwicZMaWbNdBKUrnrxDtaRhN | Nonce String                                                          |
| `X-TimeStamp`   | 1528450585                       | TimeStamp                                                             |
