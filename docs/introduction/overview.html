Integrate payments, user management, store operations, and loyalty programs via RESTful API or SDK.

Always Test in Sandbox First

Test your integration in the sandbox environment before going live — see Sandbox Testing for test cards and credentials. Contact support@revenuemonster.my to activate your sandbox account.

Before You Start​

▾
Create a Merchant Account

Sign up on the Sandbox Portal or Production Portal for your target environment.

Submit Required Documents

Complete all document submissions in the portal. Your account will be reviewed and approved by an administrator.

Get Your API Credentials

Once approved, retrieve your Client ID and Secret key from the Merchant Portal to start making API calls.

SDK vs REST API

This documentation covers the RESTful API. For language-specific SDKs, see the SDK section.
