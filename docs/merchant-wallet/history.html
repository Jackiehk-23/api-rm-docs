Check Wallet History

Method : GET
URL : https://open.revenuemonster.my/v3/wallet/history
Sandbox URL : https://sb-open.revenuemonster.my/v3/wallet/history

Check merchant wallet topup history

Request Parameters

cursorSTRING

Cursor

Example: "60"

transactionAtARRAY

Transaction At

Example: ["2021-01-11T09:54:46Z", "2021-01-11T09:54:46Z"]

referenceTypeSTRING

Reference Type

Example: "DELIVERY", "BILLING", "TOPUP_MANUAL", "TOPUP_ONLINE", "TOPUP_BANKIN""

Response Parameters

itemsOBJECT

History object

Example: (Refer to explanation below)

cursorSTRING

Cursor for next page

codeSTRING

Successfully call this endpoint. If fail, will return error code object (Refer Appendix 1: Error Codes)

Example: "SUCCESS"


History object item:
idSTRING

Wallet History ID

Example: 1598969381529317751

referenceIdSTRING

Wallet reference ID usually is your Merchant ID

Example: 4118165203679668885

walletIdSTRING

Wallet ID

Example: 1585119930335618836

typeSTRING

History type

Example: "TOPUP", "DEDUCT"

referenceTypeSTRING

History reference type

Example: "DELIVERY", "BILLING", "TOPUP_MANUAL", "TOPUP_ONLINE", "TOPUP_BANKIN"

referenceSTRING

Reference about the reference type usually it's usage information

Example: Online Transaction Topup -1598969316445167528

creditINTEGER

How much credit added or deducted

Example: 10

currentCreditINTEGER

Credit balance after added or deducted

Example: 1952

sequenceIdINTEGER

A sequential number

Example: 26

transactionAtDATETIME

When history transaction

Example: 2020-09-01T14:09:41Z
